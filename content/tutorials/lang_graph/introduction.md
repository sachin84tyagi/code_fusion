## 🆚 LangGraph vs LangChain

```
LangChain = The toolkit (models, prompts, chains, tools)
LangGraph  = The orchestration engine (workflows, state, control flow)

Think of it like:
LangChain = individual Lego pieces
LangGraph  = the instruction manual that says HOW to put them together
```
