# 5. Core Components of AI (Professional Understanding)

To truly understand AI at senior level, know these 5 pillars:

1. Data
   AI learns from data. More relevant data = better intelligence.

2. Algorithms
   Mathematical methods that find patterns in data.

3. Model
   The trained system that can make predictions.

4. Training
   Process of feeding data so the model learns patterns.

5. Inference
   When trained model makes real-world decisions.

Training = Learning phase
Inference = Using phase
